# MyFlix Movie API

Welcome to the MyFlix Movie API! This project serves as the backend for managing movie data for the MyFlix Angular client.

## About

This project is built with JavaScript and MongoDB. It provides endpoints to retrieve movie information, including details about directors, genres, and actors. The API is hosted on Heroku.

## Accessing the API

The API can be accessed at the following base URL:
[https://fierce-fortress-37859-bd3c98eebee1.herokuapp.com/](https://fierce-fortress-37859-bd3c98eebee1.herokuapp.com/)

## MongoDB Database

The movie data is stored in a MongoDB database. You can access the database and manage your collections using the MongoDB Cloud platform.

## Endpoints

### Get All Movies

Retrieves a list of all movies in the database.

### Get Movie by ID

Retrieves details of a specific movie by its ID.

### Get Director by Name

Retrieves details of a director by their name.

### Get Genre by Name

Retrieves details of a genre by its name.

### Get Actor by Name

Retrieves details of an actor by their name.

### Register User

Registers a new user.

### Login User

Logs in an existing user.

### Get User Profile

Retrieves profile information of a user by their username.

### Update User Profile

Updates profile information of a user.

### Delete User

Deletes a user account.

### Add Movie to Favorites

Adds a movie to a user's list of favorite movies.

### Remove Movie from Favorites

Removes a movie from a user's list of favorite movies.
